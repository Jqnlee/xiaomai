Premiere Pro (PR)

學習使用 Premiere Pro 進行視頻編輯的基礎知識。本課程將全面介紹時間軸編輯、特效應用、轉場設置以及如何導出視頻。不論是初學者或是希望提升技能的人，我們都能幫助你熟悉 Premiere Pro 的核心功能，從剪輯片段到添加動態效果，確保你的影視作品更加吸引人！此外，還將涵蓋格式選擇、色彩校正以及音效處理，讓作品在各種平台都能脫穎而出！

Adobe

本課程主要以圖形設計、照片編輯以及動態圖形創作為核心，為你提供全面的技能。不論你是想創建吸引人的視覺效果，還是想讓靜態設計充滿生命力，我們帶你從基礎開始，一步步深入學習這些工具的核心功能。此外，課程還會涵蓋實用的技巧與實際案例，幫助你快速將所學應用於實際項目中。

HTML

從 HTML 開始您的網頁開發旅程！學習如何構建網頁結構，添加標題、段落、圖片和鏈接。本課程為初學者設計，將帶你全面了解 HTML 的基礎知識，幫助您建立強大的網頁設計基石。我們將一步步講解如何使用 HTML 元素組織內容，使你的網站更具可讀性和實用性。此外，你還會學到如何編寫清楚且符合標準的程式碼，為未來學習 CSS 和 JavaScript 奠定良好基礎。本課程還包括實際案例和練習，讓您在動手中鞏固所學知識。現在就加入我們！

CSS

學習如何應用顏色、字體、佈局以及響應式設計，使你的網站更具視覺吸引力。本課程專為希望提升網頁設計技能的學員設計，帶你深入探索 CSS 的基礎和進階應用。課程涵蓋如何使用顏色方案增強視覺效果，以及調整字體樣式等。此外，你還會學到如何創建靈活的版面，通過系統和模型實現流暢的設計，確保網站在各種設備上的完美顯現，幫助您打造現代化、兼具美觀與功能的網站！
